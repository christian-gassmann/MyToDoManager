# MyToDoManager

Einfache Aufgaben-Management App zur Übung von Basics mir Android.

## Features
- Aufgaben erstellen, bearbeiten und löschen
- Aufgaben als erledigt markieren
- Speichern lokal

## Technologien
- Java, Android SDK
- XML-Layouts
- Git / Github
- Modularer Aufbau nach MVC
